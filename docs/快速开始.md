# kiz快速开始

如果你不方便使用电脑, 你可以在线体验kiz
[playground.kiz.random321.com](http://playground.kiz.random321.com)

## 1.下载安装包
- **下载安装包(windows)**: [#](#)
- **下载安装包(linux)**: [#](#)
- **下载安装包(mac os)**: [#](#)

## 2. 运行安装包
kiz会自动添加到系统的PATH变量中
您会在桌面得到一个kiz.exe/kiz.elf

## 3. 运行kiz.exe/kiz.elf
您看到的应该是一个这样的界面
```
This is the kiz REPL v2026.1 .
Type "help()" to get help
```

## 4. 在交互式环境中继续第一次尝试
```
This is the kiz REPL v2026.1 .
Type "help()" to get help

>>>print("Hello World")
```

如果输出
```
Hello World
```

恭喜你，执行了第一个kiz语句

## 5. 运行文件
在体验完交互式环境后

您可以尝试运行文件

在桌面创建一个`my_first.kiz`
输入
```
name = "kiz"
fn greet
    print(f"Hello from {name}")
end
greet()
```

在桌面打开Cmd, 输入
```
kiz my_first.kiz
```
您便可以看到运行结果(Cmd打印)
```
Hello from kiz
```
