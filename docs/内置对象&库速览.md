## Builtins库
### Object
- `__call__`方法：创建空对象
- `__dstr__()`方法：返回调试字符串
- `__getitem__(attr_name)`方法：返回该对象名字为atte_name的属性
- `__setitem__(attr_name, value)`方法：设置该对象名字为atte_name的属性的值为value

### Bool
- `__call__(obj)`方法：调用obj的`__bool__`方法(该方法通常返回Bool)来创建Bool对象
- `__hash__()`方法：True返回1，False返回0
- `__dstr__()`方法：返回调试字符串
### Nil
- `__hash__()`方法：永远返回0
- `__dstr__()`方法：返回调试字符串
